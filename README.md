# Project Name

A simple full-stack project with a Next.js (or Vite) frontend and a Node/NestJS backend.

## Prerequisites
- **Node.js 18+** and **npm**
- **Git**

## Quick Start

Clone the repo and run the frontend and backend in **separate terminals**.

```bash
# 1) Clone
git clone <repo-url>

# 2) Frontend
cd frontend
npm install
npm run dev

# 3) Backend (in another terminal)
cd backend
npm install
npm run start:dev
